# VEIL - Verified Ethereal Identity Ledger

## Project Overview
VEIL is a "Living Identity Firewall" for Autonomous AI Agents, featuring a biological "Brain vs. Reflexes" architecture.

## Architecture
See [Master_Architecture_v2.1.md](Master_Architecture_v2.1.md) for the detailed 7-Layer enforcement stack.

## Tech Stack
- **Reflexes (Backend):** Java 17, Spring Boot 3.2.3, Redis, PostgreSQL
- **Brain (Intelligence):** Google Gemini 3.0 Pro (via GenAI SDK)
- **Face (Frontend):** React 19, Three.js, Tailwind CSS
- **Infrastructure:** Envoy Proxy (Sidecar Pattern)
